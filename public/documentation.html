Documentation

Complete guide to using SteganoCrypt

Quick Start

Basic Usage

// Initialize Steganography
const steganography = new Steganography();

// Hide a message
const encodedImage = await steganography.encode(imageFile, message);

// Reveal a message
const decodedMessage = await steganography.decode(encodedImage);

API Reference

Methods

encode(imageFile, message)

Hides a message within an image file.

  • Parameters:
    • imageFile: File object (PNG, JPG, JPEG)
    • message: String to hide
  • Returns: Promise resolving to encoded image Blob

decode(imageFile)

Extracts a hidden message from an encoded image.

  • Parameters:
    • imageFile: File object (encoded image)
  • Returns: Promise resolving to decoded message string

Examples

Hiding a Message

// Select an image file
const imageInput = document.getElementById('image-input');
const imageFile = imageInput.files[0];

// Create message
const message = "This is a secret message";

// Hide the message
const steganography = new Steganography();
const encodedImage = await steganography.encode(imageFile, message);

// Download the encoded image
const url = URL.createObjectURL(encodedImage);
const a = document.createElement('a');
a.href = url;
a.download = 'encoded_image.png';
a.click();

Revealing a Message

// Select an encoded image
const encodedImageInput = document.getElementById('encoded-image-input');
const encodedImageFile = encodedImageInput.files[0];

// Reveal the message
const steganography = new Steganography();
const message = await steganography.decode(encodedImageFile);

// Display the message
console.log('Hidden message:', message);

Best Practices

Image Selection

  • Use PNG images for best results
  • Choose images with complex patterns or textures
  • Avoid images with large areas of solid color
  • Recommended minimum size: 800x600 pixels

Message Guidelines

  • Maximum message length depends on image size
  • Recommended: Keep messages under 1000 characters
  • Use plain text for best compatibility
  • Avoid special characters when possible

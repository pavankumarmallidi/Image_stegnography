Bug Bounty Program

Help us make SteganoCrypt more secure

About Our Bug Bounty Program

We believe in the power of community-driven security. Our bug bounty program rewards security researchers for finding and responsibly disclosing vulnerabilities in our platform.

Program Scope

In Scope

  • Web application vulnerabilities
  • API security issues
  • Cryptographic implementation flaws
  • Authentication bypasses
  • Data exposure vulnerabilities

Out of Scope

  • Social engineering attacks
  • Physical security issues
  • Denial of service attacks
  • UI/UX bugs
  • Third-party services

Reward Structure

Severity Levels

  • Critical: $1,000 - $5,000
  • High: $500 - $1,000
  • Medium: $200 - $500
  • Low: $50 - $200

Bonus Rewards

  • First-time valid reports
  • Well-documented findings
  • Suggested fixes
  • Multiple related issues

Submission Guidelines

Required Information

  • Detailed vulnerability description
  • Steps to reproduce
  • Proof of concept
  • Potential impact
  • Suggested fix (optional)

Response Time

Our Commitment

  • Initial response: 24 hours
  • Critical issues: 12 hours
  • Regular updates
  • Clear communication

Responsible Disclosure

Guidelines

  • Do not exploit vulnerabilities
  • Do not share findings publicly
  • Do not impact other users
  • Follow our disclosure policy

Hall of Fame

Top Contributors

  • Security researchers

  • Open source contributors

Legal Terms

Program Rules

  • Must comply with laws
  • No unauthorized access
  • No data exfiltration
  • No service disruption

Submit a Report

How to Report

To submit a security vulnerability, please email us at security@steganocrypt.com with the subject line "Bug Bounty Report".
